# Веб-приложение для просмотра фильмов и сериалов

Приложение для поиска, просмотра информации и изучения деталей о фильмах и сериалах. Интеграция с Kinopoisk Unofficial API предоставляет актуальную базу киноматериалов с описаниями, рейтингами, трейлерами и актёрским составом.

### **Деплой**

[https://poisk-kino.vercel.app/](https://poisk-kino.vercel.app/)

### **Локальный запуск приложения**

```bash
# Клонирование репозитория
git clone https://github.com/cayman444/movie-app.git
cd movie-app

# Установка зависимостей
npm install

# Настройка окружения
cp .env.example .env
# Добавьте ваш API ключ Kinopoisk Unofficial в .env

# Запуск в режиме разработки
npm run dev
```

### **Стек**

`React`, `TypeScript`, `Redux Toolkit`, `RTK Query`, `React Router Dom`, `Ant Design`, `Tailwind CSS`, `Vite`

### **Функциональность**

- Категории фильмов: премьеры, популярные, топ-рейтинговые
- Адаптивный интерфейс с каруселями для разных размеров экрана
- Детальные карточки фильмов с плеером, описанием, рейтингом, актёрским составом
- Фильтрация по жанрам, годам и странам, пагинация
- Поиск по фильмам с debounce
- Похожие фильмы, рекомендательная система
